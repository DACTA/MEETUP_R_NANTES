# Premier Meetup : Pourquoi R ?

Découvrez pourquoi R est l'outil indispensable pour tous projet de statistiques, data science, machine learning, big data, data visualisation, ...

*Comme c'est la première rencontre, je propose un court exposé de ce qu'est R, quels sont ses atouts et pourquoi l'utiliser en entreprise, ainsi qu'un petit aperçu de différentes possibilités offertes par ce langage.

Le but de cette première rencontre étant avant tout de faire plus ample connaissance et d'échanger ensemble autour de nos pratiques R :-)*
